The design held up ~90% (decompose.js line numbers were byte-exact despite the file growing to 899 lines). Five places diverged enough to change the build:

1. **The strip-test is EASIER than the design feared (R-XAGENT-01).** `src/render.js:renderTemplate(content, vars, modules, ideId)` already does handlebars-like `{{#if ide.X}}` stripping, and `tests/compatibility.test.js` already imports it.
